GOALZZXD

Back to projects

Project

GOALZZXD is a personal sports intelligence dashboard that transforms scattered football data into a clean, interactive platform. Designed with a bold editorial aesthetic, it allows users to follow teams, compare performance, track match statistics, and organise personalised insights through an intuitive, data-driven interface. The project combines UI/UX design, branding, information architecture, and responsive web design to make complex sports analytics more accessible and engaging.

Details

  • Scope: Identity, typography, website direction
  • Format: 1080 x 1350 project frames
  • Assets: 8 selected images
